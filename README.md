# MobiliCare

**Patient mobility tracking system — prevent pressure ulcers and immobility complications.**

A web-based MVP for nurses and clinical staff to monitor patient repositioning schedules, track bed time, and receive alerts when patients are due for repositioning.

---

## Features (Phase 1 MVP)

- Add and manage patient profiles (name, age, ward, condition, risk level)
- Track how long each patient has been in bed
- Record last repositioning time per patient
- Automatic alert status (Urgent / Due soon / On schedule)
- Dashboard with live patient monitoring
- Reports view with compliance and risk analytics
- One-click "Repositioned" logging per patient

---

## How to use

No installation needed. This is a single HTML file.

1. Download `index.html`
2. Open it in any modern browser
3. Or deploy it to Netlify / Vercel / GitHub Pages for a shareable link

---

## Deploy in 60 seconds (Netlify)

1. Go to [netlify.com](https://netlify.com) and sign up free
2. Drag and drop `index.html` onto the deploy area
3. You get a live link instantly — share it with your team
